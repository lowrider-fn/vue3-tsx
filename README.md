# Vue3-tsx. [Demo](https://lowrider-fn.github.io/vue3-tsx/)

## Команды
```bash
# старт проекта
yarn start

# сборка проекта
yarn build

# запуск eslint
yarn lint

# запуск stylelint
yarn stylelint

# проверка типов
yarn types

# переустанавливает зависимости
yarn reinstall
```
